========================
The tricks behind Magica
========================

This book includes all design documents for the features that make up
Magica, as well as the threat models we consider. This is intended for
a more technical audience, security auditors, and developers using
Magica who wish to have a deeper understanding of its guarantees,
performance characteristics, and risks.
